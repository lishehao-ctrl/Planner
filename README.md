# One-Screen Day (Step 1 Scaffold)

Single-screen "today action sequence" planner scaffold.

## Stack
- Vite + React + TypeScript
- TailwindCSS
- React Router
- Zustand
- Dexie (IndexedDB)
- Vitest + React Testing Library
- ESLint + Prettier

## Run
```bash
npm install
npm run dev
npm run test
npm run lint
```

## Current Routes
- `/` Today page placeholder
- `/settings` Settings page placeholder

## Folder Structure
- `src/app/` router + app shell
- `src/pages/` route pages
- `src/ui/` UI placeholder components
- `src/domain/` placeholder types
- `src/state/` Zustand placeholder store
- `src/storage/` Dexie placeholder DB
- `src/styles/` Tailwind entry stylesheet

## Scope of Step 1
Scaffold only. No business logic yet.
